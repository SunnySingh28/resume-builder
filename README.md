# Resume Builder

A modern and ATS-friendly Resume Builder built with React, Vite, Tailwind CSS, html2canvas, and jsPDF. Create professional resumes, choose from multiple templates, preview instantly, and export high-quality PDFs with clickable links.

## Features

### Resume Creation

* Personal Information
* Profile Photo Upload
* Professional Summary
* About Me Section
* Education Details
* Work Experience
* Projects
* Skills
* Languages
* Achievements

### Multiple Resume Templates

* Professional Template
* Modern Template
* Classic Template
* Minimal Template
* LaTeX Resume Template

### Template Highlights

* ATS-Friendly Resume Layouts
* Professional Two-Column Design
* LaTeX-Inspired Academic Resume Format
* Profile Photo Support
* Responsive Resume Preview
* Clean Typography and Formatting

### Clickable PDF Links

Exported PDFs preserve clickable links for:

* Email
* Phone Number
* LinkedIn
* GitHub
* Portfolio
* LeetCode
* Codeforces
* Project Repository Links

### PDF Export

* One-Click PDF Download
* High-Quality PDF Generation
* A4 Resume Layout
* Resume Preview Before Download
* Clickable Hyperlinks in Exported PDF

### ATS Optimization

* ATS-Friendly Templates
* Structured Resume Sections
* Professional Formatting
* Improved Readability

### Local Storage

* Auto-Save Resume Data
* Data Persists After Refresh
* Reset Resume Option

```bash
cd resume-builder
npm install
npm run dev
```
